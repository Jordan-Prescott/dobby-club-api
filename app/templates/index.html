Welcome To The Dobby Club API

It is a RESTful API dedicated to the UK sitcom Peep Show that provides data about characters, episodes, quotes and more. Links to the GitHub repository, documentation, and live examples are provided below.

GitHub Docs Try it Live

Get Started

To start using the API, check out our documentation and try out some example endpoints:

  • /characters - Get a list of characters
  • /episodes - Get a list of episodes
  • /quotes - Get a list of quotes
